# Proyecto Final - Red Social Académica

## Descripción

Sistema desarrollado en C++ para la gestión de una red social académica utilizando estructuras de datos avanzadas.

## Estructuras Implementadas

* Tabla Hash
* Árbol AVL
* Lista Enlazada
* Pila
* Grafo no dirigido
* BFS
* DFS

## Funcionalidades

* Carga de estudiantes desde CSV
* Registro manual de estudiantes
* Búsqueda y eliminación de estudiantes
* Gestión de proyectos
* Conexiones entre estudiantes
* Ranking académico
* Estadísticas de estructuras
* Benchmark en C++
* Benchmark comparativo en Java
