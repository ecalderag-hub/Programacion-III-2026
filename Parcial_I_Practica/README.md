- Breve explicación de cómo funciona el programa:
Este programa utiliza punteros en C++ para crear tres nodos de forma dinámica en memoria usando la instrucción new. Cada nodo tiene un espacio para guardar un número entero y un puntero que sirve para conectarlo con el siguiente nodo. El programa le pide al usuario que ingrese un valor para cada nodo y luego los enlaza en el mismo orden en que fueron ingresados, formando una lista enlazada simple.

Después, el programa recorre los nodos utilizando punteros para acceder a los datos almacenados y mostrarlos en la consola junto con el nombre completo del estudiante. Finalmente, se utiliza la instrucción delete para liberar la memoria de cada nodo, lo cual es importante para evitar el desperdicio de memoria.
